# SmartPortal

SmartPortal is a data-driven education platform designed to analyze student performance and identify academic risks at an early stage.

## Overview

In many schools, student data is scattered and difficult to analyze. Teachers often lack tools to track student progress and identify those who are falling behind.

SmartPortal solves this problem by providing a centralized system that transforms academic data into clear and actionable insights.

## Features

- Dashboard with student performance analytics  

- Early Warning System for identifying at-risk students  

- Self-study section with additional materials  

- Progress and achievement tracking  

## How It Works

The platform collects student assessment data and converts it into percentage-based performance indicators.

Based on predefined thresholds, the system identifies students who are at risk and need additional support. Teachers can then assign extra materials such as quizzes or video lessons.

## Technology Stack

- Frontend: HTML, CSS, JavaScript  

- Backend: Node.js  

- Data processing logic for performance analysis  

## How to Run

### Frontend

Open the `index.html` file in your browser.

### Backend (optional)

1. Install dependencies:

   npm install

2. Run the server:

   node server.js
